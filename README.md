# Brewlish Mystery — 潮騒館の最後の編集

10日間で事件の真相へ到達する、資料探索型のデジタル本格ミステリーです。

## プレイを始める

GitHub Pagesの公開URLから作品トップを開いてください。Dayサイトで物語と探索を進め、「今日の問い」に正解すると Today's Brew Code が表示されます。そのコードをBrewPhoneへ入力すると、新しい捜査資料が解放されます。

- 作品トップ: `/`
- BrewPhone: `/brewphone/`
- Dayサイト: `/day01/`〜`/day10/`

## 推奨環境

スマートフォンまたはPCの最新版ブラウザでプレイできます。進行状況、既読、捜査ノートはブラウザ内に保存されます。
