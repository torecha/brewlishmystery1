# Brewlish Mystery — 潮騒館の最後の編集

10日間で事件の真相へ到達する、資料探索型のデジタル本格ミステリーです。

## プレイを始める

GitHub Pagesの公開URLから作品トップを開いてください。各Dayは「物語と現場探索 → BrewPhoneで必読資料を調査 → 今日の問い → その日の調査終了」の順で進みます。現場探索を終えると Today's Brew Code が表示され、そのコードをBrewPhoneへ入力すると新しい捜査資料が解放されます。

- 作品トップ: `/`
- BrewPhone: `/brewphone/`
- Dayサイト: `/day01/`〜`/day10/`
- エンディング: `/ending/`（Day 10完了後に解放）

## 収録内容

- BrewPhone資料104件（全文聴取、検視・ログ・写真・メールを含む）
- 作中Web 16サイト／83ページ
- Recorder音声再現、証拠映像、画像内の文字を起点にしたWeb探索
- BrewPhone内で発生する緊急通知（対応端末では振動、非対応端末では画面振動と通知音）

## 推奨環境

スマートフォンまたはPCの最新版ブラウザでプレイできます。進行状況、既読、捜査ノートはブラウザ内に保存されます。
